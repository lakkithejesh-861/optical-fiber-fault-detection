# AI-Based Optical Fiber Fault Detection in High-Speed Communication Networks

## About
An AI-powered system to automatically detect and locate faults in optical
fiber communication networks using machine learning algorithms.

## Technologies Used
- Python
- MATLAB
- Machine Learning (scikit-learn)
- ESP32 Microcontroller
- Optical Fiber Communication concepts

## Features
- Detects fault types: bends, breaks, connector faults
- Analyses optical signal loss patterns
- Classifies fault location automatically
- Implemented on ESP32 hardware
